## ページ操作

| キー | アクション |
| ---- | ---- |
| Home | ページの先頭へ移動 |
| END | ページの最後へ移動 |
| Ctrl + R | ページ再読み込み |

## ウィンドウ操作

| キー | アクション |
| ---- | ---- |
| Ctrl + N | 新しいウィンドウを開く |
| Ctrl + Shift + T | シークレットモードで新しいタブを開く |

## タブ操作

| キー | アクション |
| ---- | ---- |
| Ctrl + T | 新しいタブを開く |
| Ctrl + W | アクティブタブを閉じる |
| Ctrl + Tab | 次のタブへ移動 |
