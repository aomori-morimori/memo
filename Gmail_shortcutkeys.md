## 次/前のメール参照

| キー | アクション | 補足 |
| ---- | ---- | ---- |
| J | 受信トレイで下へ移動 <br> メール参照時には過去のメールへ移動 | 古いメールの参照 |
| K | 受信トレイで上へ移動 <br> メール参照時には新しいメールへ移動 | 新しいメールの参照 |

## メール選択

| キー | アクション | 補足 |
| ---- | ---- | ---- |
| X | 受信トレイでメールを選択 | チェックボックスへのチェック |

## 検索

| キー | アクション | 補足 |
| ---- | ---- | ---- |
| / | 検索ボックスがアクティブになる | |

## 取り消し

| キー | アクション | 補足 |
| ---- | ---- | ---- |
| Z | 最後に行ったアクションを取り消す | Ctrl + Z のような感じ |

## 既読/未読

| キー | アクション | 補足 |
| ---- | ---- | ---- |
| Shift + I | 選択したメールを既読にする | |
| Shift + U | 選択したメールを未読にする | |

## 重要マーク

| キー | アクション | 補足 |
| ---- | ---- | ---- |
| + or = | 重要マーク付与 | |
| - | 重要マーク削除 | |
| S | 重要なメールにスターを付与/削除 | |
